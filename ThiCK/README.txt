CÁC BƯỚC CHỈNH SỬA CODE THI CUỐI KÌ:
C Basic có trọng số 40-60 nên với 5,5 điểm quá trình và 3 điểm cuối kì bạn sẽ vừa đủ 4 điểm qua môn.
Để được 3 điểm thì cần menu đúng (1đ) khai báo đúng CTDL (1đ) giải phóng bộ nhớ cấp phát động (1đ)

Nếu chỉ cần 4 điểm cuối kì thì làm theo các bước sau:
1. Sửa số lượng tính năng và tên tính năng theo đề
2. Sửa các thuộc tính của elmType và node_t trong file tree.h
3. Sửa filename trong file ThiCK.c thành file đề cho (ví dụ: data.txt)
Có thể để lại các comment để vấn đáp. (các dòng có // ở đầu là comment)

Nếu muốn điểm cao hơn 3 thì làm thêm các bước sau:
1. Sửa hàm compare trong tree.c (CTRL H và thay ID thành thuộc tính so sánh của đề (ví dụ: tên, MSSV, điểm,...)) 
2. Thêm code vào các hàm tinhnang của file ThiCK.c (phần này mới ăn điểm nên cố lên 👍)
3. Sửa dòng 11 của file traversal.c theo định dạng đề yêu cầu. Hàm inorderprint sẽ in ra cây theo thứ tự từ điển, do đó cần sửa hàm printf để in ra các thuộc tính đề yêu cầu.

Lúc thi đừng gọi mình vì mình sẽ không cầm điện thoại đọc tin nhắn hay nghe các bạn gọi đâu.
Chúc các bạn thi tốt!
